Operadores de comparação 
------------------------

Igual (==) -> verifica se os valores são iguais

Idêntico (===) -> verifica se os valores são iguais e do mesmo tipo

Diferente (!=) -> verifica se os valores são diferentes

Não idêntico (!==) ->  verifica se os valores são diferentes ou os tipos também 

Menor (<) ->  verifica se o valor da esquerda é menor do que o valor da direita

Maior (<) ->  verifica se o valor da direita é menor do que o valor da esquerda

Menor ou igual (<) ->  verifica se o valor da esquerda é menor do que o valor da direita ou igual

Maior ou igual (<) ->  verifica se o valor da direita é menor do que o valor da esquerda ou igual


Operadores lógicos
------------------

E (&&) -> verdadeiro se todas as expressões ou valores forem verdadeiros
OU (||) -> verdadeiro se pelo menos uma das expressões ou valores forem verdadeiros 
Negação (!) -> inverte o resultado da expressão de comparação
